## 快捷键
- 主页(index): `i`
- 栏目(section): `s`
- 主题(theme): `t`
- 热门(hot): `h`
- 日期选择(date): `d`
- 关闭页面(close): `c`
- 查看评论(comments message): `m`
- 切换评论(switch): `w`
- 夜间模式切换(night mode): `n`
