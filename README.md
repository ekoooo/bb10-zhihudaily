# 知乎日报 BB10

> 暂时屏幕适配的是passport, 其他bb10机器没有测试

## 完成功能
- 首页, 置顶新闻(轮播图部分), 每日新闻
- 栏目以及查看栏目内容
- 主题以及查看主题内容
- 查看评论(点击评论部分进入)
- 今日热门消息
- 往日新闻查看
